## 命名规则

各平台维护一个 `XX软件清单.md` 文件，按功能分类（开发工具 / 多媒体 / 办公效率 / 系统维护 等）分段记录。
新增软件直接追加到对应分类下，不再单独建文件。

安装包存放于 `G:\软件\`，本清单记录软件信息与用途说明。
