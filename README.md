## 四指令使用

| 指令 | 功能 | 示例 |
|:---|:---|:---|
| 📝 `记：` | 保存到收件箱 | `记：发现一个好用的截图工具 Snipaste` |
| 🔍 `查：` | 查询知识库 | `查：装机清单` `查：福州美食` |
| 📦 `整：` | 整理收件箱 | `整：收件箱` |
| 📋 `列：` | 浏览目录 | `列：` `列：01-软件工具` |

## 三端数据流

```
┌─────────┐   QQ聊天    ┌──────────┐   Git SSH    ┌──────────┐
│ 手机iOS  │ ────────→  │ OpenClaw │ ──────────→ │ GitHub   │
└─────────┘            │ (耗子助手)│              │ 私有仓库  │
                       └──────────┘ ←───────────  └──────────┘
                                              ↑ git clone/pull
                                              │
                                        ┌──────────┐
                                        │  你的电脑  │
                                        └──────────┘
```

- **手机**：QQ 发送"记/查/整/列"指令
- **OpenClaw（VM）**：MCP 工具读写第二大脑 → git push/pull
- **电脑（本地）**：git pull 同步，VSCode 编辑
