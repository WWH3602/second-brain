## 3. 三端数据流

```
┌─────────┐   QQ聊天    ┌──────────┐   Git pull/push   ┌──────────┐
│  手机(iOS) │ ────────→ │  OpenClaw │ ──────────────→ │  GitHub   │
└─────────┘            │  (耗子助手) │                 │  私有仓库  │
                        └──────────┘ ←──────────────  └──────────┘
                                              ↑ git clone/pull
                                              │
                                        ┌──────────┐
                                        │  你的电脑  │
                                        │(AI赋能中心)│
                                        └──────────┘
```

- **手机**：只发 QQ 消息（iOS 无需装 Git App）
- **OpenClaw（VM）**：pull → 写文件 → commit → push，仓库路径 `/home/wwh/second-brain`
- **电脑（本地）**：直接 VSCode 编辑，OpenClaw 定时 pull 同步
