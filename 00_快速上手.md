## 故障排查速查表

| 现象 | 原因 | 解决 |
|------|------|------|
| QQ 发消息没反应 | Hook 没配 | 按 `04_OpenClaw_QQ_Hook_配置.md` 配 |
| `git push` 失败 | Token 过期 | 重新生成 PAT + 更新 `02_凭据与安全.md` |
| `git pull` 冲突 | 本机和 VM 同时改一个文件 | **不要**两边同时改 → 用 00-Inbox 隔离 |
| 找不到 `02_凭据与安全.md` | 文档没创建 | 看 `02_凭据与安全.md` 模板，照抄一份 |
| 凭据误推到 GitHub | .gitignore 失效 | 立刻 Revoke Token + 删除历史 commit |
